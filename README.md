# 📖 About

**C Programming Labs** is a structured learning repository dedicated to mastering the C programming language through comprehensive notes, clean source code, hands-on practice, and practical projects.

The repository follows a topic-based approach, allowing each concept to be learned independently while progressively building a strong understanding of programming fundamentals, memory management, and problem-solving.

Every topic includes theory, examples, implementation, and organized source code to create a professional learning archive.


# ⚙️ Compile & Run

## GCC

Compile

```bash

```

Run

```bash
./program
```
